・Hello, World!を表示するサンプルプログラムを稼働させるための環境構築の途中である。
　・利用するのはgnu-efi、QEMUのみとした。
　・Makefileを作成し、可能な限り作業の自動化を図る。
　　→　現在、イメージ作成までの自動化を完了したが、実際に稼働させるところの途中である。
　・ディレクトリのイメージとしては以下の通り
　　src ---- loader : ブートローダの開発部。この中でdisk.imgの作成までを行う。
         |
         --- kernel : カーネルの開発部。時宜に応じてドライバ用作業スペースなども作成予定。ソースのみベタ置きし、他のファイルは以下のようにまとめる。
         |     |
         |     --- include : インクルードフォルダのまとめ場所。
         |     |
         |     --- obj     : 生成されるオブジェクトファイルの置き場所。
         |     |
         |     --- bin     : カーネルの実行ファイルの置き場所。
         |
         --- runenv : 実行環境。loader内でmake runenvまでを完了することで環境が整う。
　・今の所Makefileのコマンドによって環境を構築しているが、エラー時の対応が面倒なこともあるので、シェルスクリプトに置き換えも検討する。
　・次回は、QEMUによって、ハードディスク、UEFIの環境から起動できるようにコマンドを整備する。
